EagerCentroid

Companion object EagerCentroid

class EagerCentroid extends MutableWeightedVector with Serializable

A mutable weighted vector that eagerly adds new vectors or subtracts vectors to a form centroid. Consequently, it is appropriate for use with dense vectors.
