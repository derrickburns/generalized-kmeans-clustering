  3.  class RandomIndexEmbedding extends Embedding

    Embeds vectors in high dimensional spaces into dense vectors of low dimensional space in a way that preserves similarity as per the Johnson-Lindenstrauss lemma.

    Embeds vectors in high dimensional spaces into dense vectors of low dimensional space in a way that preserves similarity as per the Johnson-Lindenstrauss lemma.

    This technique is called random indexing.

    We use a family of universal hash functions to avoid pre-computing the random projection.

    See also

    introduction to random indexing

  4.  class SymmetrizingEmbedding extends Embedding

    One can create a symmetric version of any Bregman Divergence

    One can create a symmetric version of any Bregman Divergence

    This one is

    distance(x,y) = KL(x,y) + KL(y,x) + (1/2) ||x-y||2 + (1/2) || gradF(x) - gradF(y)||2

    The embedding is simply

    x => x + gradF(x) (Lemma 1 with alpha = beta = 1)
